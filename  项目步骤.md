# 1 创建项目

打开终端，使用`create-react-app`创建项目

```bash
npx create-react-app ygshop
```

# 3. 安装对css 预处理器sass 支持

```
npm install node-sass --dev
```

# 4.添加移动端的适配方案

> Flex 布局 + rem + flexible+sass

1. 暴露webpack配置，即 react-scripts 包

```js
npm run eject
```

⚠️ 在运行该命令的时候，要先将已经修改的文件提交到本地仓库中过，否则会报错！
